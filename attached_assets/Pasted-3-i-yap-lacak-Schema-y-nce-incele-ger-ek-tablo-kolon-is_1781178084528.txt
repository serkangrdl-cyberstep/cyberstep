── 1.1 MERSIS Şirket Yöneticisi Araması ─────────

export async function enrichFromMersis(
  companyName: string,
  domain: string
): Promise<{ officerName: string | null; title: string | null }>

MERSIS açık arama endpoint'i:
GET https://mersis.gtb.gov.tr/portal/rest/firma/ara?firmaAdi={companyName}&kayitDurumu=1

- companyName: domain'den türet (örn: "diktas.com.tr" → "diktas")
- Sonuçtan ilk eşleşen şirketin yönetici adını çek
  (yetkiliListesi veya benzeri alan)
- timeout: 10sn, hata olursa null döndür, fırlat değil
- User-Agent header ekle (bot engelini aşmak için):
  "Mozilla/5.0 (compatible; CyberStep-Enrichment/1.0)"

Döndür:
{ officerName: "Ahmet Yılmaz", title: "Genel Müdür" }
veya
{ officerName: null, title: null }

NOT: MERSIS rate limit belirsiz — istekler arası 2sn sleep ekle.
Günde max 50 sorgu ile başla (cron'da maxLeads: 50 parametresi).

── 1.2 KAP Yönetici Araması ─────────────────────

export async function enrichFromKap(
  companyName: string
): Promise<{ officerName: string | null; email: string | null; title: string | null }>

KAP açık API:
GET https://www.kap.org.tr/tr/api/basicSearchByName?name={companyName}

- İlk eşleşen şirketin member kodu al
- GET https://www.kap.org.tr/tr/api/memberInfo/{memberCode}
  → yöneticiler listesi: ad, unvan, e-posta (varsa)
- E-posta varsa döndür, yoksa sadece ad + unvan
- timeout: 8sn, hata → null döndür

NOT: KAP sadece halka açık/borsaya kote şirketler için çalışır.
Sonuç gelmezse sessizce null döndür — KOBİ'lerin çoğunda
veri olmayacak, bu beklenen durum.

export async function enrichLeadFromTrSources(

── 2.1 LinkedIn Arama URL Üretici ───────────────

Lead tablosunun her satırına "LinkedIn'de Ara" butonu ekle.

URL formatı (yeni sekmede açılır):

Şirket adı biliniyorsa:
https://www.linkedin.com/search/results/people/?keywords={şirketAdı}&titleKeyword=IT+OR+CTO+OR+Genel+Müdür+OR+Bilgi+İşlem+OR+CISO+OR+Dijital

Sadece domain biliniyorsa:
https://www.linkedin.com/search/results/companies/?keywords={domain}

officer_name biliniyorsa (MERSIS/KAP'tan geldi):
https://www.linkedin.com/search/results/people/?keywords={officerName}+{şirketAdı}

Şirket adını şu yerden türet (öncelik sırasıyla):
1. lead_candidates.company_name (varsa)
2. domain'den: "diktas.com.tr" → "Diktaş" (ilk segment, büyük harf)

Buton tasarımı:
- LinkedIn mavi rengi (#0A66C2)
- LinkedIn "in" ikonu (SVG inline)
- Tooltip: "LinkedIn'de karar verici ara"
- Tek tıkla yeni sekmede aç (target="_blank")

## Kontakt Zenginleştirme Kaynakları ve Başarı Oranları

| Kaynak | TR KOBİ Başarı Oranı | Veri Türü |
|--------|---------------------|-----------|
| WHOIS/RDAP | ~%15-20 | E-posta |
| Web scraping | ~%25-30 | E-posta |
| MERSIS | ~%40-50 | Ad + Unvan |
| KAP | ~%5-10 | Ad + E-posta (borsalı) |
| Hunter.io | ~%5-10 | E-posta |
| Apollo.io | ~%8-12 | E-posta |
| LinkedIn (ISR Manuel) | ~%60-70 | Profil + E-posta |

Not: Apollo.io ve Hunter.io Türk KOBİ veritabanı
kapsamı düşük. Kontakt bulunamayan leadler için
LinkedIn Sales Navigator manuel araması ISR tarafından
yapılmaktadır.

## ISR LinkedIn İş Akışı

1. Admin panel → Lead Discovery → "Kontakt Eksik" filtresi aç
2. Her lead için "LinkedIn'de Ara" butonuna tıkla
3. MERSIS'ten gelen yetkili adı varsa kişi araması yap
4. Bulunan e-postayı "Manuel Gir" ile kaydet
5. Tüm qualified leadler için CSV export → Sales Navigator import
